# Mã Giảm Giá

Website tổng hợp mã giảm giá, coupon khuyến mãi — xây dựng bằng PHP.

## 📋 Tính năng

- Hiển thị danh sách mã giảm giá
- Tìm kiếm coupon
- Giao diện responsive

## 🛠️ Công nghệ

- **Backend:** PHP
- **Frontend:** HTML, CSS, JavaScript

## ⚙️ Cài đặt

Deploy lên Apache/Nginx web server với PHP enabled.

```bash
git clone https://github.com/trucuit/magiamgia.git
cd magiamgia
```
