# Mã Giảm Giá

Website tổng hợp mã giảm giá, coupon khuyến mãi từ Lazada, Shopee, Tiki, Sendo,...

## 🌐 Demo

**Live:** [https://trucuit.github.io/magiamgia/](https://trucuit.github.io/magiamgia/)

## 📋 Tính năng

- Hiển thị danh sách mã giảm giá theo danh mục
- Coupon mua sắm, du lịch, taxi
- Tìm kiếm coupon
- Form đăng ký nhận khuyến mãi
- Giao diện responsive (Bootstrap)

## 🛠️ Công nghệ

- **Frontend:** HTML, CSS, JavaScript
- **Framework:** Bootstrap 4
- **Icons:** Font Awesome

## 📁 Cấu trúc

```
magiamgia/
├── index.html     # Trang chính (static)
├── css/           # Stylesheets
├── js/            # JavaScript
├── image/         # Hình ảnh, logo
├── images/        # Hình ảnh bổ sung
├── fonts/         # Font files
└── html/          # HTML partials
```
